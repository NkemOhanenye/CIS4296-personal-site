Projects

I've worked on some neat things.
Unit Assessment System (UAS)

In a group environment encompassing a whole year, a system is being developed to replace the way Assessments are done and reviewed for the Temple University Office of Institutional Research and Assessment. The project is based off the Student Learning Management Assessment System (SLAM) and through the development we will learn the Software Development Life Cycle (SDLC) as getting used to interviewing clients and document our understandings and findings into a user manual.

Mastermind Game

In a group environment encompassing of 2 members, we developed a game of choice in C# using Visual Studio focusing on encaspulation and low coupling. Through the development of the game we learned how to create a search algorithm to check what the user chose as the code and compared it with the randomly generated code and show the user if they were close or not. The game's window also dynamically changed size depending on the difficulty of the game the user wanted to play.

Quadforge

Woring in a group of programmers in configuring a Raspberry Pi with Java to read a motor and upload the readings to a text file. Through the project we learned how to read pin outs and adjust the settings of the motor, for example: adjusting the speed at which the motor would spin.
